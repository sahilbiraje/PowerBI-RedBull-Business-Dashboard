# 🚀 Red Bull Business Analytics Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive Power BI dashboard analyzing Red Bull's business performance using financial and marketing data.

The dashboard enables users to explore company performance through KPIs, revenue trends, regional performance, market share, marketing investments, and quarterly sales.

---

## 📊 Dashboard Features

- Executive KPI Cards
- Revenue Trend Analysis
- Revenue by Region
- Market Share Analysis
- Marketing Spend Analysis
- Quarterly Sales Dashboard
- Interactive Year Slicer
- Dynamic Filtering

---

## 🛠 Tools Used

- Microsoft Power BI
- Power Query
- DAX
- Microsoft Excel
- Data Modeling

---

## 📈 KPIs Included

- Total Revenue
- EBITDA
- Cans Sold
- Employees
